The retrieval paths (where the document lives)
Path 1 — This page (easiest): bookmark this URL. Load → Download as Word or .json.

Path 2 — Straight from storage (last resort): Cloudflare dashboard → Storage & Databases → KV → open the namespace book-storage (primary) or book-storage-backup (backup) → the key book:dr-johnson holds the full document as text. Snapshots are the snap:dr-johnson:<date> keys in the backup namespace.

Path 3 — Off-site copy (GitHub, if enabled): open your backup repo on GitHub → file backups/dr-johnson.json. It holds the full document, and GitHub's History for that file is a complete, independent version history you can view or download anytime.
